# <a name="azure-analysis-services-samples"></a>Azure Analysis Services-exempel
Använd följande exempelresurser för att du lär dig mer om och testa Analysis Services för din miljö.

## <a name="code-samples"></a>Kodexempel
Den [Analysis Services](https://github.com/Microsoft/Analysis-Services) databasen på GitHub innehåller kodexempel för öppen källkod och community-projekt. 

## <a name="tabular-model-project-and-database-samples"></a>Tabellmodell projekt och databas-exempel
[Adventure Works för Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) är det vanligaste exemplet tabellmodellprojekt på GitHub. Du kan hämta ett VS-projekt eller en tabellmodelldatabas med färdiga exemplet.


## <a name="sample-databases-on-github"></a>Exempeldatabaser på GitHub
Dessa Exempeldatabaser på GitHub kan användas för att skapa och testa ditt eget modeller. 

[Adventure Works Exempeldatabaser](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2014)

[Wide World Importers Exempeldatabaser](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Adventure Works Internet Sales exempel modell i Azure-portalen
Om du har en Analysis Services-server i Azure-portalen kan du snabbt och skapa enkelt en exempelmodell. I serverns översiktssidan i Azure-portalen, klickar du på **nya modellen**, och sedan i **Välj en datakälla**väljer **exempeldata**.
