# <a name="azure-analysis-services-samples"></a>Azure Analysis Services exempel

Använd följande exempel resurser för att få hjälp att lära dig om och testa Analysis Services för din miljö.

## <a name="code-samples"></a>Kodexempel

[Analysis Services](https://github.com/Microsoft/Analysis-Services) -lagringsplatsen på GitHub innehåller kod exempel för öppen källkod och community-projekt. 

## <a name="tabular-model-project-and-database-samples"></a>Projekt med tabell modell och databas exempel

[Adventure Works för Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) på GitHub är det vanligaste exemplet tabell modell projekt. Du kan ladda ned ett VS-projekt eller en slutförd exempel tabell modell databas.

## <a name="sample-database-on-azure"></a>Exempel databas på Azure

Azure Synapse Analytics (tidigare SQL Data Warehouse) innehåller ett exempel på en AdventureWorksDW-databas som kan ingå i en etablerad resurs. Mer information finns i [snabb start: skapa och fråga en dedikerad SQL-pool i Azure Synapse Analytics ](/azure/sql-data-warehouse/create-data-warehouse-portal).

## <a name="sample-databases-on-github"></a>Exempel databaser på GitHub

Dessa exempel databaser på GitHub kan användas för att skapa och testa dina egna modeller. 

[Exempel databaser för Adventure Works](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks)

[Exempel databaser för Wide World-importörer](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Exempel modell för Adventure Works Internet Sales i Azure Portal

Om du har en Analysis Services-server i Azure Portal kan du snabbt och enkelt skapa en exempel modell. På sidan Översikt för servern i Azure Portal klickar du på **ny modell** , och i **Välj en data källa** väljer du **exempel data**.
