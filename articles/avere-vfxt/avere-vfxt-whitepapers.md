Resurserna i den här tabellen innehåller fallstudier, teknik White Paper-faktablad, informationsgrafik och annat innehåll som kan hjälpa dig att förstå hur Avere vFXT för Azure passar in i ditt arbetsflöde. 

## <a name="deploy-agisoft-photoscan-on-azure"></a>Distribuera Agisoft PhotoScan på Azure

**Länk:** [Distribuera Agisoft PhotoScan på Azure med Avere vFXT för Azure eller BeeGFS](https://azure.microsoft.com/mediahandler/files/resourcefiles/deploy-agisoft-photoscan-on-azure-with-azere-vfxt-for-azure-or-beegfs/AgiSoft%20PhotoScan%20on%20Azure%20using%20Avere%20vFXT%20or%20BeeGFS.pdf) 

Den här guiden innehåller stegvisa anvisningar för att installera Agisoft PhotoScan photogrammetry programvara som backas upp av Avere vFXT lagring eller BeeGFS parallellt filsystem. Författare kan Paulo Marques da Costa av AzureCAT, visar hur du ställer in PhotoScan på Azure Virtual Machines (VM). Lagring med höga prestanda går det snabbare att bearbetningstid och resultatet av sitt benchmark-testerna ingår. Den här miljön kan skalas upp och ned efter behov och stöder terabyte lagring utan att offra prestanda.

## <a name="hybrid-cloud-for-file-based-workloads"></a>Hybridmoln för fil-baserade arbetsbelastningar

**Länk:** [Informationsgrafik: Att skapa ett hybridmoln för fil-baserade arbetsbelastningar](https://azure.microsoft.com/resources/building-a-hybrid-cloud-for-file-based-hpc-workloads/)

Använd Avere vFXT för Azure för att utöka beräkningskapacitet med din befintliga infrastruktur på plats eller flytta data och databehandling i molnet på basis av tillfälligt eller permanent.  

## <a name="cloud-bursting"></a>”Cloud bursting”
