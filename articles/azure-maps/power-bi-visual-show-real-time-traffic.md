# <a name="show-real-time-traffic"></a>Visa real tids trafik

Trafik skikts funktionen täcker real tids trafik data ovanpå kartan. Om du vill aktivera den här funktionen flyttar du skjutreglaget för **trafik nivå** i rutan **format** till **på** position. Detta kommer att täcka trafikflödes data som färgkodade vägar.

> [!div class="mx-imgBorder"]
> ![En karta som visar trafik data i real tid](media/power-bi-visual/traffic-layer.png)

Följande inställningar är tillgängliga i avsnittet **trafik skikt** .

| Inställningen         | Beskrivning    |
|-----------------|----------------|
| Visa incidenter  | Anger om trafik incidenter, t. ex. väg stängningar och konstruktion, ska visas på kartan. |
| Trafik kontroll | Lägger till en knapp till kartan som gör det möjligt för rapport läsare att aktivera eller inaktivera trafik skiktet.  |
