# <a name="get-started-with-the-azure-quickstart-center"></a>Kom igång med Azure snabb start Center

Azure snabb starts Center är en guidad upplevelse i Azure Portal som är tillgänglig för alla som vill förbättra sina kunskaper om Azure. För organisationer som är nya för Azure är det snabbaste sättet att publicera och konfigurera din moln miljö.

## <a name="overview"></a>Översikt

Azure snabb starts Center har tre alternativ för att komma igång:

* **Installations guider**: Våra guider är utformade för IT-administratören och moln arkitekten och introducerar viktiga begrepp för Azure-antagande. Strukturerade steg hjälper dig att vidta åtgärder när du lär dig att använda Microsofts rekommenderade metod tips. I Migreringsguiden får du hjälp att utvärdera beredskap och planera när du förbereder dig för att byta till Azure.

* **Starta ett projekt**: Om du är redo att skapa en resurs kan du använda det här avsnittet för att lära dig mer om dina val innan du genomför ett tjänst alternativ. Du lär dig mer om tjänsten och varför du bör använda den, utforska kostnader och identifiera krav. När du har gjort ditt val kan du gå direkt till skapa.

* **Online-utbildning**: I det här avsnittet av Azure snabb starts Center beskrivs kostnads fria introduktions kurser från Microsoft Learn. Välj en panel för att starta en kurs och lär dig mer om moln koncept och hantera resurser i Azure.

## <a name="how-to-use-azure-quickstart-center"></a>Använda Azure snabb starts Center

1. Logga in på [Azure Portal](https://portal.azure.com).
2. Välj **alla tjänster** i det övre avsnittet av marginal listen.
1. Välj **allmän** > **snabb starts Center**. Välj stjärn ikonen för att spara till dina **Favoriter**.
