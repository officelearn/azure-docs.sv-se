# <a name="create-services-included-with-azure-free-account"></a>Skapa tjänster som ingår i kostnadsfritt Azure-konto

Kostnadsfritt Azure-konto innehåller en begränsad mängd ledigt tjänster i tolv månader. Mer information finns i [vad är Azure gratiskonto](https://azure.microsoft.com/free/free-account-faq/). Azure håller reda på din användning av varje tjänst och debitera inte dig om du överskrider gränserna. I dessa gränser ger kostnadsfritt Azure-konto dig flexibilitet att använda olika konfigurationer för att uppfylla dina behov. 

## <a name="create-free-services-in-the-azure-portal"></a>Skapa gratistjänster i Azure-portalen
Vi rekommenderar att du använder den [ledigt webbtjänstsida](https://go.microsoft.com/fwlink/?linkid=859151) att skapa nya tjänster. Du kan logga in till den [Azure-portalen](http://portal.azure.com)väljer **alla tjänster** till vänster och sökning på **kostnadsfria tjänster**. 

## <a name="services-can-be-created-in-any-region"></a>Tjänster kan skapas i en region
Så länge som du är inom gränserna som kan du skapa tjänster kostnadsfritt i en region där tjänster är tillgängliga. Till exempel, får du 750 timmars ledigt B1S Windows-dator varje månad med kostnadsfritt Azure-konto. Du kan skapa den virtuella datorn i en region där B-serien virtuella datorer är tillgängliga. Azure debitera inte dig om du överskrider 750 timmar. En kund i USA kan etablera en virtuell dator med Windows B1S i västra Europa och använda 750 timmar kostnadsfritt. 

Läs om Azure-tjänstetillgänglighet per region i [produkter som är tillgängliga efter region](https://azure.microsoft.com/regions/services/).

## <a name="multiple-instances-of-services-can-be-created-within-allowed-limits"></a>Flera instanser av tjänster kan skapas överskrider den tillåtna gränsen
Du kan skapa flera instanser av tjänster kostnadsfritt som summan av användning är inom användningsgränsen på. Till exempel, får du 750 timmars ledigt B1S Windows-dator varje månad med din kostnadsfritt Azure-konto. Du kan använda 750 timmar i en kombination som du vill använda. Du kan till exempel skapa 5 B1S Windows-datorer och använda dem för 150 timmar. 

## <a name="need-help-contact-support"></a>Behöver du hjälp? Kontakta supporten

Om du behöver hjälp, [supporten](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) få snabbt lösa problemet.
