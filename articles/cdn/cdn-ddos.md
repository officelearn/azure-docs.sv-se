# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS Protection

Ett Content Delivery Network ger DDoS skydd genom design. Utöver den globala kapaciteten för att absorbera volym attacker har Azure CDN ytterligare DDoS skydd som beskrivs nedan, utan extra kostnad.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN från Microsoft

Azure CDN från Microsoft skyddas av [Azure Basic DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Den integreras i Azure CDN från Microsoft-plattformen som standard och utan extra kostnad. Full skala och kapacitet för Azure CDN från Microsofts globalt distribuerade nätverk ger skydd mot vanliga attacker mot nätverks nivå genom att alltid övervaka trafik övervakning och i real tid. Det grundläggande DDoS-skyddet skyddar också mot de vanligaste, ofta förekommande, lager 7 DNS-frågor och lager 3-och 4-mätkolvar som riktar sig mot CDN-slutpunkter. Den här tjänsten har även en beprövad spårnings post för att skydda Microsofts företags-och konsument tjänster från storskaliga attacker.

## <a name="azure-cdn-from-verizon"></a>Azure CDN från Verizon

Azure CDN från Verizon skyddas av Verzion: s patentskyddade DDoS-skydds plattform. Den integreras i Azure CDN från Verizon som standard och utan extra kostnad. Det ger grundläggande skydd mot de vanligaste, ofta förekommande, ofta förekommande lager 7 DNS-frågor och nivå 3-och 4-radioattacker som riktar sig mot CDN-slutpunkter.

## <a name="azure-cdn-from-akamai"></a>Azure CDN från Akamai

Azure CDN från Akamai skyddas av Akamai: s DDoS-skydds plattform. Den integreras i Azure CDN från Akamai som standard och utan extra kostnad. Det ger grundläggande skydd mot de vanligaste, ofta förekommande, ofta förekommande lager 7 DNS-frågor och nivå 3-och 4-radioattacker som riktar sig mot CDN-slutpunkter.
