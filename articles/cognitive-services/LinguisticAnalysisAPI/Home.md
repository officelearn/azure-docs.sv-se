# <a name="what-is-the-linguistic-analysis-api-preview"></a>Vad är API för språkanalys? (Förhandsversion)

> [!IMPORTANT]
> Förhandsversionen av Språkanalys upphörde den 9 augusti 2018. Vi rekommenderar att du använder [Azure Machine Learning-textanalysmoduler](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics) för textbearbetning och -analys.

API:et för språkanalys ger åtkomst till verktyg för bearbetning av naturliga språk (NLP) som identifierar strukturen av text. Den aktuella versionen innehåller tre typer av analys:

- [Uppdelning i meningar och tokenisering](Sentences-and-Tokens.md)
- [Del av tal-märkning](POS-tagging.md)
- [Parsning av valkrets](Constituency-Parsing.md)

## <a name="getting-started"></a>Komma igång

När du har bekantat dig med de huvudtyper av analyser som anges ovan kan du titta på slutpunkten för att [hämta listan med tillgängliga analysverktyg](AnalyzersMethod.md) och hur de [namnges](Analyzer-Names.md) och sedan slutpunkten för att [analysera nya indata](AnalyzeMethod.md).
