# <a name="natural-language-support-for-azure-cognitive-services"></a>Stöd för naturligt språk för Azure Cognitive Services

Azure Cognitive Services kan du skapa program som se, höra, tala med och förstå dina användare. Mellan dessa tjänster, mer än tre dussin språk stöds, så att användarna kan kommunicera med ditt program på ett naturligt sätt.

Den här artikeln är uppdelad i två delar. Först listar core språk brukar ha stöd för Azure Cognitive Services. Det andra avsnittet fokuserar på ytterligare språkstöd för varje tjänst. Tillgängliga språk kan variera beroende på land/region och marknaden.

## <a name="core-languages"></a>Core-språk

Dessa core språk stöds i Azure Cognitive Services:

* Kinesiska
* Svenska
* Franska
* Tyska
* Italienska
* Japanska
* Korean¹
* Portugisiska
* Spanska

> [!NOTE]
> ¹ tal till Text och Video Indexer stöds inte.

## <a name="additional-language-availability-by-service"></a>Ytterligare Språktillgänglighet av tjänsten

Dessa tabeller markerar tillgängliga språk genom tjänstekategori. Core-språk som undantas. Använd länkarna nedan om du vill veta mer om stöd för ytterligare språk och land/region och marknaden tillgänglighet.
