# <a name="data-catalog-developer-samples"></a>Data Catalog developer-exempel
Börja utveckla Data Catalog-appar med hjälp av Data Catalog REST-API: et. Data Catalog REST-API: et är ett REST-baserad API som ger programmatisk åtkomst till Data Catalog-resurser för att registrera, anteckna och söka i datatillgångar programmässigt.

Här följer de exempel som vi har på GitHub:

* [Kom igång med Azure Data Catalog](https://azure.microsoft.com/resources/samples/data-catalog-dotnet-get-started/)
  
  Ett exempel som visar hur du autentiserar med Azure AD för att registrera, söka och ta bort en datatillgång med Data Catalog REST-API: et.
* [Massinläsning registrera och kommentera](https://azure.microsoft.com/resources/samples/data-catalog-dotnet-excel-register-data-assets/)
  
  Ett exempel som visar hur du bulk registrera datatillgångar från en Excel-arbetsbok med Data Catalog REST API: et och Open XML.
* [Import/Export-verktyget](https://azure.microsoft.com/resources/samples/data-catalog-dotnet-import-export/)
  
  Ett exempel som visar hur du använder Data Catalog REST-API: et för att hämta tillgångar från Azure Data Catalog och serialisera dem till en fil. Den visar också hur du tar en uppsättning tillgångar serialiserad som JSON och skicka dem till katalogen. Det har stöd för export av en delmängd av katalogen med hjälp av en sökfråga.

* [Massinläsning import ordlistetermer](https://azure.microsoft.com/resources/samples/data-catalog-bulk-import-glossary/)

    Detta exempel visar hur du importerar ordlistetermer från CSV-filer till ADC ordlista.
