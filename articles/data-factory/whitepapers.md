# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory-faktablad 

White Paper-faktablad kan du utforska Azure Data Factory på en djupare nivå. Den här artikeln innehåller en lista över tillgängliga whitepapers för Azure Data Factory.

| **White Paper** | **Beskrivning** |
| --- | --- |
|[Azure Data Factory – integrering i molnet](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Det här dokumentet beskriver hur Azure Data Factory gör det möjligt att skapa ett modernt informationslager, aktivera avancerad analys att skapa smarta SaaS-program och flytta SQL Server integration Services-paketen till Azure.|
|[Migrering av data från en lokal relationella datalager till Azure med Azure Data Factory](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | Det här dokumentet adresser komplexiteten för att migrera tiotals TB data från befintliga lokala relationsinformationslager (till exempel Netezza, Oracle, Teradata och SQLServer) med hjälp av Azure Data Factory till Azure (till exempel Blob Storage eller Azure Data Lake Storage). Utmaningar och metodtips illustreras runt flexibilitet, prestanda, skalbarhet, hantering och säkerhet för stordata inmatning resa till Azure med Azure Data Factory. |
|[Azure Data Factory: SSIS i molnet](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Det här dokumentet går över varför du skulle vilja migrera dina befintliga SSIS-arbetsbelastningar till Azure Data Factory och adress vanliga överväganden och frågor. Vi sedan vägleder dig genom de tekniska detaljerna för att skapa en Azure-SSIS IR och sedan hur du kan ladda upp, köra och övervaka dina paket Azure Data Factory med hjälp av de verktyg du är förmodligen är bekant med som SQL Server Management Studio (SSMS). |
