# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Vilka verktyg ingår i Azure-Data Science Virtual Machine?

Data Science Virtual Machine är det enklaste sättet att utforska data och göra maskin inlärning i molnet. Data vetenskaps Virtual Machines är förkonfigurerade med det fullständiga operativ systemet, säkerhets korrigeringar, driv rutiner och populär program vara för data vetenskap och utveckling. Du kan välja maskin varu miljö, som sträcker sig från CPU-koncentriska datorer med lägre kostnad till mycket kraftfulla datorer med flera GPU: er, NVMe-lagring och stora mängder minne. För datorer med GPU: er installeras alla driv rutiner, alla Machine Learning-ramverk är versions matchade för GPU-kompatibilitet och acceleration är aktiverat i alla program program som stöder GPU: er.

Den Data Science Virtual Machine levereras med de mest användbara verktygen för data vetenskap som är installerade:
