# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Vilka verktyg ingår i Azure Data Science Virtual Machine?

Data Science Virtual Machine är ett enkelt sätt att utforska data och göra maskin inlärning i molnet. Data vetenskaps Virtual Machines är förkonfigurerade med det fullständiga operativ systemet, säkerhets korrigeringar, driv rutiner och populär program vara för data vetenskap och utveckling. Du kan välja maskin varu miljö, som sträcker sig från CPU-koncentriska datorer med lägre kostnad till mycket kraftfulla datorer med flera GPU: er, NVMe-lagring och stora mängder minne. För datorer med GPU: er installeras alla driv rutiner, alla Machine Learning-ramverk är versions matchade för GPU-kompatibilitet och acceleration är aktiverat i alla program program som stöder GPU: er.

Den Data Science Virtual Machine levereras med de mest användbara verktygen för data vetenskap som är förinstallerade.

## <a name="how-to-choosecondaenvironment"></a>Så här väljer du Conda-miljö  

> [!NOTE]
> Det har uppstått ändringar när du växlar till den nya versionen av Data Science Virtual Machine från den gamla versionen. 

**Växla från Windows Server 2016 till Windows Server 2019** </br> 

* Conda-miljön har angetts till python37_default. Vi har inte stöd för python2. Om du använder TensorFlow eller pytorch i din python Conda-miljö använder du py37_tensorflow eller py37_pytorch. Om du bara använde AzureML använder du azureml_py36_automl </br>

* Om du använder TensorFlow eller pytorch i din AzureML Conda-miljö använder du azureml_py36_tensorflow eller azureml_py36_pytorch. </br>

**Växlar från Ubuntu 16,04 till Ubuntu 18,04** </br>

* Om du använder TensorFlow eller pytorch i din azureml_py36 Conda-miljö använder du azureml_py36_tensorflow respektive azureml_py36_pytorch. </br>

* Om du använder TensorFlow eller pytorch i din py37 Conda-miljö använder du py37_tensorflow eller py37_pytorch. </br>

* Annars använder du python37_default eller om du bara använde azureml_py36 använda azureml_py36_automl </br>
