# <a name="walkthroughs-executing-the-team-data-science-process"></a>Genomgångar av köra Team av vetenskapliga data

Dessa **slutpunkt till slutpunkt genomgång** visar stegen i teamet vetenskap av data för specifika scenarier. De visar hur du kombinerar moln, lokala verktyg och tjänster i ett arbetsflöde eller en rörledning för att skapa en **intelligent program**. Genomgångar grupperas efter **plattform** som de använder.


## <a name="walkthrough-descriptions"></a>Genomgång beskrivningar

Här följer kortfattade beskrivningar av exemplen genomgång ange på deras respektive plattformar:


- [Azure Machine Learning arbetsstationen](walkthroughs-aml-with-tdsp.md) dessa genomgången självstudier och exempel visar hur du skapar Team datavetenskap strukturerad projekt med Azure Machine Learning arbetsstationen för att göra naturligt språk för databehandling och klassificering.
- [HDInsight Spark genomgång med hjälp av PySpark och Scala](walkthroughs-spark.md) dessa genomgång använda PySpark och Scala på en Azure Spark-kluster för att göra förutsägelseanalyser.
- [HDInsight Hadoop genomgång med Hive](walkthroughs-hdinsight-hadoop.md) dessa genomgång använda Hive med HDInsight Hadoop-kluster för att göra förutsägelseanalyser.
- [Azure Data Lake genomgång med hjälp av U-SQL](walkthroughs-azure-data-lake.md) dessa genomgång med U-SQL Azure Data Lake för att göra förutsägelseanalyser.
- [SQL Server](walkthroughs-sql-server.md) dessa genomgång använda SQL Server, SQL Server R Services och SQL Server Python Services för att göra förutsägelseanalyser.
- [SQL Data Warehouse](walkthroughs-sql-data-warehouse.md) dessa genomgång använder SQL Data Warehouse för förutsägelseanalys.
