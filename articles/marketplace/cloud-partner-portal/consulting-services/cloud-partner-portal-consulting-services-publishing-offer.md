# <a name="azure-and-dynamics-365-consulting-service-offer"></a>Azure och Dynamics 365 konsulttjänst erbjuder

Den här artikeln förklarar hur du publicerar en konsult tjänsterbjudande till antingen Microsoft <a href="https://azuremarketplace.microsoft.com">Azure Marketplace</a> eller <a href="https://appsource.microsoft.com">AppSource</a>. Lösningar baserade på Microsoft <a href="https://dynamics.microsoft.com">Dynamics 365</a>, <a href="https://powerbi.microsoft.com">Power BI</a>, och <a href="https://powerapps.microsoft.com">PowerApps</a> kan publiceras på AppSource. Andra erbjudanden baserat på andra Microsoft- <a href="https://azure.microsoft.com/services">Azure-tjänster</a> kan visas på Azure Marketplace.

## <a name="publishing-benefits"></a>Publicera fördelar

Publicering till antingen Microsoft-marknadsplatser har följande fördelar:

- Marknadsför ditt företag genom att utnyttja Microsoft varumärke.
- Potentiellt nå ut till fler än 100 miljoner Office 365 och Dynamics 365-användare på AppSource och nå ut till fler än 200 000 organisationer via Azure Marketplace.
- Få hög kvalitet leads från dessa marknadsplatser.
- Hämta dina befordras av Microsoft-fältet och telefonförsäljning team-tjänster.

## <a name="publish-a-consulting-service-offer"></a>Publicera en konsult tjänsterbjudande

Det här avsnittet beskrivs element för att publicera en konsult tjänsteerbjudanden. Publicering är uppdelad i följande huvuddelar:

- [Consulting service krav](./cpp-consulting-service-prerequisites.md) listor kraven för att publicera konsulter erbjuder för:
 
    - Dynamics 365 för Customer Engagement 
    - Dynamics 365 för & ekonomichef 
    - Dynamics 365 Business Central 
    - Power BI 
    - PowerApps
- [Skapa ett nytt erbjudande](./cpp-consulting-service-create-offer.md) visas de steg som krävs för att skapa en konsult service erbjudandet post med hjälp av Cloud Partner Portal:
    - [Definiera inställningar för erbjudandet](./cpp-consulting-service-define-offer-settings.md).
    - [Ange information i butiken och om du vill publicera på Azure Marketplace eller på AppSource.](./cpp-consulting-service-storefront-details.md)
- [Publicera ditt erbjudande](./cpp-consulting-service-publish-offer.md) förklarar hur du skickar in erbjudande för publicering.
