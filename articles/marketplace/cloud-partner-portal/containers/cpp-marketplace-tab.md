# <a name="container-marketplace-tab"></a>Behållaren Marketplace-fliken

Den **Marketplace** fliken den **nytt erbjudande** sidan kan du ge din potentiella kunder marknadsföring, försäljning och juridisk information och avtal och hantera leads som genereras från den Marketplace. Lägga till din marketing-specifikt innehåll i den **översikt** avsnittet.

## <a name="overview"></a>Översikt

I det här avsnittet ska ange du allmän information om dina Azure Marketplace erbjuder.  En asterisk (*) läggs till fältnamnet anger att det krävs.

![Marketplace-informationen](./media/containers-marketplace-tab.png)

I följande tabell beskrivs syftet med och innehållet i de här fälten. Required fields are indicted by an asterisk (*).

|    Fält                  |       Beskrivning                                                            |
|  ---------                |     ---------------                                                          |
| **Rubrik\***                 | Rubrik för erbjudandet. Det ska visas på en framträdande plats i marketplace.  Maxlängden är 50 tecken. |
| **Sammanfattning\***               | Kort sammanfattning av erbjudandet. Maximal längd är 100 tecken. |
| **Lång sammanfattning\***          | Längre sammanfattning av erbjudandet (trots att den kan vara samma som den **sammanfattning**).  Maximal längd är 256 tecken. |
| **Beskrivning\***           | Beskrivning av erbjudandet.  Maximal längd är 3000 tecken, stöder enkel HTML-formatering.
| **Identifierare för marknadsföring**  | En unik URL att koppla till det här erbjudandet, som vanligtvis innehåller namnet på din organisation och lösning. Maxlängden är 50 tecken.   |
| **Förhandsversion av prenumerations-ID: N\*** | Lägg till en till 100 prenumeration identifierare av förhandsgranskningar. Dessa white visas prenumerationer har åtkomst till erbjudandet när den publiceras innan det lanseras. |
|  |  |
