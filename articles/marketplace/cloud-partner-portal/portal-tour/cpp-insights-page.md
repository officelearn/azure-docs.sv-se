# <a name="insights-page"></a>Sidan med insikter

Den [ **Insights** ](https://cloudpartner.azure.com/#insights) sidan innehåller följande uppsättningar instrumentpaneler för att hjälpa dig att förstå och ökar försäljningen av dina marketplace-erbjudanden.  Varje instrumentpanel kommer åt via en flik med samma namn överst på sidan.


|  **Instrumentpanel**    |  **Visade innehåll**               |
|  -------------    |  ---------------------               |
| **Sammanfattning**       | diagram, trender och värdena för aggregerade data som sammanfattar marknadsplatsaktivitet utgivarens erbjudanden |
| **Payout**        | payouts och relaterade transaktioner i grafiska och nedladdningsbara format |
| **Beställningar och användning** | beställningar och information om användningen i grafiska och nedladdningsbara format |
| **Kunden**      | kundinformation, inklusive deras inköpschef profil |
| **Distribution**    | lyckade och misslyckade distributionsinformation i både grafiskt och Händelsenivå format |
| **Nedladdningar**     | lista över hämtningsbegäranden under de senaste 30 dagarna |
| **Analys**     | Sammanfattning av web-analys och analys av Kampanjresultat |
|  |  |


## <a name="summary-dashboard"></a>Instrumentpanel för sammanfattningar
Sammanfattning av instrumentpanelen är den första fliken och standard-fliken visas när du navigerar till analyssidan.  Den här avsiktliga designen visar att den här instrumentpanelen ger utgivare bredaste vy över försäljningsaktivitet för sina erbjudanden.  Det ger grafiska representationer av den normaliserade informationen om användningstrenden; all användning varje månad; populära användning av geografiska marknaden, erbjudande och kund. och betalnings trend.  Utdata kan anpassas genom att ange start- och slutdatumen för analys.  Följande bild visar ett exempel av instrumentpanel för sammanfattning för Contoso för juni till och med augusti 2018.
