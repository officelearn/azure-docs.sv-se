# <a name="monitor-encoding-job-progress-with-the-azure-portal"></a>Övervaka förloppet för kodnings jobbet med Azure Portal

> [!NOTE]
> Inga nya funktioner läggs till i Media Services v2. <br/>Kolla in den senaste versionen [Media Services v3](https://docs.microsoft.com/azure/media-services/latest/). Se även [vägledning för migrering från v2 till v3](../latest/migrate-from-v2-to-v3.md)

## <a name="overview"></a>Översikt

När du kör jobb krävs ofta ett sätt att spåra jobbets förlopp. 

Klicka på **Inställningar** (överst på sidan) för att övervaka förloppet för kodningsjobbet och välj sedan **Jobb**.

![Jobb](./media/media-services-portal-vod-get-started/media-services-jobs.png)

Du kan klicka på jobbet för att se mer information.

![Jobb](./media/media-services-portal-vod-get-started/media-services-job-progress2.png)

## <a name="next-steps"></a>Nästa steg
När kodnings jobbet är ett kan du publicera och spela upp till gångar, enligt beskrivningen [här](media-services-portal-publish.md).
