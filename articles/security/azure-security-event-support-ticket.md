# <a name="how-to-log-a-security-event-support-ticket"></a>Så här loggar du ett supportärende för händelse

1. Gå till [supportinformation för utgivaren](https://support.microsoft.com/en-us/getsupport?wf=0&tenant=ClassicCommercial&oaspworkflow=start_1.0.0.0&locale=en-us&supportregion=en-us&pesid=16230&ccsid=636450758943226673) och logga in med ditt Microsoft-autentiseringsuppgifter.
2. Välj ”säkerhetshändelse” som problemtyp och välja mellan ”Security Incident” och ”säkerhetsproblem” kategorier.

    ![Händelsetyp och Definition][1]

3. När du väljer typ av Problem och kategori, klickar du på den ”**förfrågan om att starta**-knappen. Ange följande information för att hjälpa oss att bättre förstå problemet.

    i. Vad är problemet och/eller säkerhetsproblem?

    ii. Efter säkerhetsrisker, ange CVE (mitre.org) eller den fyllda ut CVSS3 v3 Kalkylatorn (https://www.first.org/cvss/calculator/3.0).

    III. Finns det en lösning eller minimering? Om Ja, bör du ange åtgärdssteg.

    IV. Har du ett meddelande som du vill skicka till kunder? Vi arbetar med att skapa ett meddelande visas om det är tillämpligt.

4. Bekräftelse - när du har skickat din fråga kan vi bekräfta mottagandet inom en arbetsdag och tilldela problemet en prioritet och allvarlighetsgrad.

    - Om du vill kommunicera med oss om problemet Använd bekräftelsenummer i all korrespondens.
    - Du kan visa förloppet på ditt problem när som helst.

5. Vad händer nu? Beroende på problemet och allvarlighetsgrad vidtas följande steg:

    - Vi kommer att informera resultatet av vår utvärdering till dig. Beroende på resultatet, kan vi ta bort eller begära att du ändrar ditt erbjudande. I detta fall kan arbetar vi med dig för att säkerställa att avbrott för berörda kunder minimeras.
    - Vi arbetar med dig för att minska effekten av incident/sårbarheten för våra gemensamma kunder.
