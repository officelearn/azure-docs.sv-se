# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

Microsoft Security Response Center (MSRC) leds av några av världens mest erfarna säkerhetsexperter. De identifiera, övervaka, svara, och lösa säkerhetsincidenter och lokala och molnbaserade sårbarheter runt klockan, varje dag på året.

Förutom arbetet kontinuerlig har MSRC-teamet ett antal resurser som är tillgängliga för dig så att du kan förstå hur du skyddar dina Azure-tillgångar och distributioner mer effektivt.

## <a name="the-msrc-blog"></a>MSRC-blogg

Den [MSRC-blogg](https://blogs.technet.microsoft.com/msrc/) kan du gå till att få de senaste nyheterna på vad MSRC gör för att skydda mot molnhot.

## <a name="white-papers"></a>White paper-faktablad

MSRC har publicerat ett antal [white paper-faktablad](https://technet.microsoft.com/library/bb969102.aspx) som hjälper dig att du förstår vad de gör och hur de gör. Vissa ger insikt i hur vi skyddar Microsoft cloud och innehåller användbar information om hur du kan använda samma säkerhetskonfigurationer.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Security forskare engagemanget och glädje program

MSRC stöder samarbete och relationer med säkerhetsforskare globalt för att gå vidare säkerhet för Microsoft-produkt.

Microsoft-program som glädje betala forskare för:

- Nya utnyttjande tekniker
- Skydden idéer som minskar Informationsarkitektur kryphål
- Identifiering av kritiska säkerhetsrisker i Microsoft lokala och molnbaserade program.

Mer information om MSRC finns på den [MSRC-startsidan](https://technet.microsoft.com/library/dn440717.aspx).
