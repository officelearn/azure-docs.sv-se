# <a name="azure-security-best-practices-and-patterns"></a>Azure säkerhetsmetoder och mönster
Vi har för närvarande följande Azure säkerhetsmetoder och mönster artiklar. Se till att den här webbplatsen regelbundet för att se uppdateringar till vår växande lista över Azure säkerhetsmetoder och mönster:  

* [Säkerhetsmetoder för Azure-nätverk](azure-security-network-security-best-practices.md)
* [Metodtips för säkerhet för Azure data och kryptering](azure-security-data-encryption-best-practices.md)
* [Kontrollera säkerhetsmetoder Identitetshantering och åtkomst](azure-security-identity-management-best-practices.md)
* [Sakernas Internet säkerhetsmetoder](azure-security-iot-best-practices.md)
* [Azure IaaS säkerhetsmetoder] (azure-security-iaas.md)
* [Azure gräns säkerhetsmetoder](../best-practices-network-security.md)
* [Implementera en säker hybridnätverksarkitektur i Azure](../guidance/guidance-iaas-ra-secure-vnet-hybrid.md)
* [Azure PaaS Best Practices] (https://docs.microsoft.com/en-us/azure/security/security-paas-deployments)

Azure tillhandahåller en säker plattform som du kan skapa dina lösningar. Vi kan tillhandahålla tjänster och teknik för att göra dina lösningar på Azure säkrare. Många av er har Tonande intresse för vad Microsoft rekommenderar som bästa praxis och mönster för att förbättra säkerheten på grund av de många alternativ tillgängliga för dig.

Vi förstår ditt intresse och har skapat en samling dokument som beskriver vad du kan göra aktuella rätt kontexten att förbättra säkerheten i Azure-distributioner.

I dessa metodtips och mönster artiklarna diskuterar vi en samling med bästa praxis och användbara mönster efter specifika avsnitt. Dessa bästa praxis och mönster härleds från våra upplevelser med dessa tekniker och erfarenheter från kunder som dig själv.

För varje rekommenderar vi strävar efter att förklara:

* Vad som är bästa praxis
* Varför du vill aktivera den bästa praxis
* Vad kan vara resultatet om du inte aktivera bästa praxis
* Möjliga alternativ till bästa praxis
* Hur du kan lära dig att aktivera bästa praxis

Vi hoppas att inklusive många fler artiklar om Azure säkerhetsarkitekturen och bästa praxis. Om det finns avsnitt som du vill att vi ska inkludera kan berätta för oss i området diskussion längst ned på den här sidan.
