# <a name="microsoft-trust-center"></a>Microsoft Säkerhetscenter
Information om Azure-webbplats på Azure.com ger dig den information du behöver för att planera, utforma, distribuera, konfigurera och hantera molnlösningar på ett säkert sätt. Med Microsoft Trust center kan har du också den information du behöver att vara säker på att Azure-plattformen som du kör dina tjänster är säker.

Vi vet att när du låta dina program och data till Azure ska du har frågor. Var finns det? Vem som har åtkomst till den? Vad gör Microsoft för att skydda den? Hur kan du kontrollera att Microsoft gör det står?

Och vi har svar. Eftersom den är dina data måste du bestämma vem som har åtkomst och du samarbeta med oss att bestämma var de finns. Vi använder den senaste säkerhetsteknik och världsklass kryptografi för att skydda dina data. Vår efterlevnad oberoende granskas och vi är transparent för många nivåer – från hur vi hanterar juridiska krav på din kundinformation för säkerheten i vår kod.

Här är vad du hitta på Microsoft Trust Center:

* [Säkerhet](https://aka.ms/tcsecurity) – Lär dig hur Microsoft Cloud-tjänsterna är skyddade.
* [Sekretess](https://aka.ms/tcprivacy) – förstå hur Microsoft garanterar säkerheten för dina Data i Microsoft-molnet.
* [Kompatibilitet](https://aka.ms/tccompliance) – identifiera hur Microsoft hjälper organisationer att uppfylla nationell, regional och branschspecifika krav för insamling och användning av data för enskilda användare har.
* [Genomskinlighet](https://aka.ms/tctransparency) – visa hur Microsoft anser som du hanterar dina data i molnet och hur Microsoft hjälper dig så mycket som möjligt om hur informationen hanteras.
* [Produkter och tjänster](https://aka.ms/tcproductsservices) – visa alla Microsoft Cloud-produkter och tjänster på en plats
* [Förtroende Tjänstportalen](https://aka.ms/tcservicetrportal) – få kopior av oberoende granskningsrapporter för Microsofts molntjänster, riskbedömningar, säkerhet metodtips och relaterade material.
* [Vad är nytt](https://aka.ms/tcwhatsnew) – ta reda på vad är nytt i Microsoft Cloud-förtroende
* [Resurser](https://aka.ms/tcresources) – Undersök faktablad, videor och fallstudier på Microsoft betrodda moln

Den [Microsoft Trust Center](https://www.microsoft.com/trustcenter) har vad du behöver att förstå vad vi göra för att skydda Microsoft-Cloud.
