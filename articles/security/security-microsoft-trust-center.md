# <a name="microsoft-trust-center"></a>Microsoft Säkerhetscenter
Säkerhetsinformation för Azure-plats på Azure.com ger dig den information du behöver för att planera, utforma, distribuera, konfigurera och hantera dina molnlösningar på ett säkert sätt. Med Microsoft Trust center kan ha du också information du behöver för att vara säker på att Azure-plattformen där du kör dina tjänster är säker.

Vi vet att när du överlåter dina program och data till Azure ska du har frågor. Var är den? Vem som har åtkomst till den? Vad gör Microsoft för att skydda den? Hur kan du verifiera att Microsoft gör vad den ska innehålla?

Och vi har svar. Eftersom det är dina data kan du bestämma vem som har åtkomst och du samarbeta med oss för att bestämma var de finns. Vi använder den senaste säkerhetsteknik och i världsklass kryptografi för att skydda dina data. Anger vår efterlevnad granskas oberoende av varandra och vi är transparent på många nivåer – från hur vi hanterar juridiska förfrågningar om kunddata till säkerheten i vår kod.

Här är vad du hittar vid Microsoft Trust Center:

* [Security](https://aka.ms/tcsecurity) – Lär dig hur Microsoft Cloud-tjänsterna är skyddade.
* [Sekretess](https://aka.ms/tcprivacy) – förstå hur Microsoft garanterar sekretessen för dina Data i Microsoft-molnet.
* [Efterlevnad](https://aka.ms/tccompliance) – identifiera hur Microsoft hjälper organisationer att uppfylla nationella, regionala och branschspecifika krav för insamling och användning av enskilda personers data.
* [Transparens](https://aka.ms/tctransparency) – visa hur Microsoft anser att du kontrollera att dina data i molnet och hur Microsoft hjälper dig så mycket som möjligt om hur dessa data ska hanteras.
* [Produkter och tjänster](https://aka.ms/tcproductsservices) – se alla Microsoft Cloud-produkter och tjänster på samma ställe
* [Service Trust Portal](https://aka.ms/tcservicetrportal) – och få kopior av oberoende granskningsrapporter för Microsoft cloud services, riskbedömningar, säkerhet metodtips och relaterade material.
* [Vad är nytt](https://aka.ms/tcwhatsnew) – ta reda på vad är nytt i Microsoft Cloud-förtroende
* [Resurser](https://aka.ms/tcresources) – Undersök white paper-faktablad, videoklipp och fallstudier av betrodda Microsoft-molnet

Den [Microsoft Trust Center](https://www.microsoft.com/trustcenter) har allt du behöver förstå vad vi gör för att skydda Microsoft Cloud.
