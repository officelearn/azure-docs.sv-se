# <a name="service-bus-messaging-samples"></a>Asynkrona Service Bus-exempel

Asynkrona Service Bus-exempel som visar viktiga funktioner i [Service Bus-meddelanden](https://azure.microsoft.com/services/service-bus/). För närvarande kan hittar du exemplen på två platser:

- [Service Bus-meddelanden exempel på GitHub](https://github.com/Azure/azure-service-bus/tree/master/samples/DotNet): en nyare uppsättning exempel som finns på GitHub. Finns det [Readme-filen](https://github.com/Azure/azure-service-bus/blob/master/samples/DotNet/Microsoft.ServiceBus.Messaging/README.md) i lagringsplatsen beskrivningar av dessa .NET-exempel. I prover uppdateras kontinuerligt, så kom tillbaka ofta efter uppdateringar.
- [MSDN-exempel sidan](https://code.msdn.microsoft.com/site/search?query=service%20bus&f%5B0%5D.Value=service%20bus&f%5B0%5D.Type=SearchText&ac=5): äldre exempel som finns i MSDN code gallery. Även om de här exemplen fortfarande fungerar, bevaras inte och kan vara inaktuella i förhållande till aktuell programming rekommendationer.
 
## <a name="service-bus-explorer"></a>Service Bus Explorer

Dessutom kan den [Service Bus Explorer](https://github.com/paolosalvatori/ServiceBusExplorer) är ett exempel finns på GitHub där du kan ansluta till ett namnområde för Service Bus-tjänsten och enkelt hantera meddelandeentiteter. Verktyget ger avancerade funktioner, till exempel import/export- och möjlighet att testa meddelandeentiteter och relay-tjänster. Du hittar fullständig Service Bus Explorer käll- och dokumentation på [GitHub](https://github.com/paolosalvatori/ServiceBusExplorer).
