# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple för molnet lösningar Provider Program: vanliga frågor och svar


## <a name="overview"></a>Översikt
Följande är frågor och svar som du kan ha som en StorSimple-partner när du distribuerar eller hantera en virtuell StorSimple-matris på Azure-portalen.

Frågor och svar ordnas i följande kategorier:

 - [StorSimple och molnet lösningar Provider Program (CSP)](#storsimple-and-csp-model)
 - [Distribuera och hantera virtuella matris som en partner](#deploy-and-manage-storsimple-as-a-partner)
 - [Stöd för StorSimple-partners för Kryptografiprovider](#support-for-storsimple-partners)

## <a name="storsimple-and-csp-model"></a>StorSimple- och CSP-modellen

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Vad innebär det att ha StorSimple som en del av programmets molnet lösningar Provider (CSP)?
 StorSimple för CSP gör det möjligt för våra CSP partner att sälja virtuella StorSimple-matris till sina kunder. Den här modellen ger CSP-partners att äga livscykeln för slutpunkt till slutpunkt kunden med direkta etablering, fakturering och stöd för Microsofts molntjänster. 

Mer information finns på [StorSimple i CSP program](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Är StorSimple en användningsbaserad eller baserat på klient-licenser som Office 365 och Enterprise Mobility Suite (EMS)?
StorSimple är en användningsbaserad tjänst. Kunder som skaffa virtuella StorSimple-matris via CSP vägen betala för användning av virtuella matris och Azure, debiteras som separata-poster i CSP faktura. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>Liknar affärsmodell StorSimple som andra tjänster som Office 365 och Enterprise Mobility Suite? 
Ja. Modellen är en Partihandel rabatt precis som andra användningsbaserad tjänster i Azure. Partihandel rabatt för StorSimple fungerar på samma sätt som för andra Azure-tjänster. Den här modellen ger våra CSP-partner möjlighet att sälja mer och enheten Azure-förbrukningen via virtuella StorSimple-matris.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Vilka StorSimple-SKU: er är tillgängliga vid lanseringen?
På samma SKU: er som är tillgängliga med fristående StorSimple-tjänster. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Är det någon skillnad mellan StorSimple köpt direkt från Microsoft under EA och StorSimple som en del av CSP?
Nej. Kunder kan också köpa andra tjänster som erbjuds av CSP-partner tillsammans med StorSimple virtuell matris (SVA) under en faktura under CSP. Observera att för upphandling den fysiska enheten för StorSimple 8000-serien kunder fortfarande behöver gå igenom EA vägen. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>I vilket marknader finns StorSimple för CSP vid lanseringen?
StorSimple virtuell matris för CSP är tillgänglig på alla marknader där StorSimple finns idag. Mer information finns i [lista över regioner där StorSimple är tillgängligt](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Vilken typ av lösningar kan en partner leverera med StorSimple- och CSP? 
Det finns flera lösningar:

- Partners kan sälja virtuella StorSimple-matris. 
- Partners kan leverera lösningar som är uppbyggd kring StorSimple och andra Azure-tjänster eller program från andra tillverkare. Mer information finns i några möjliga [lösningar som finns runt StorSimple](https://aka.ms/storsimple-build-solutions).
- De kan även identifiera nya modeller för företag med hanterade tjänster.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Vad är uppmuntra tillgänglig för StorSimple CSP partners? Jag kvalificera för uppmuntra alla CSP-programmet?
För information om uppmuntra, gå till [CSP programmet uppmuntra ](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>Distribuera och hantera StorSimple som en partner

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hur administrera virtuella StorSimple-matris på CSP? 
Du kan lägga till virtuella StorSimple-matris prenumerationer kundernas kontot via Partner Center. Du kan dessutom använda Azure-portalen för att lägga till användare i prenumerationerna.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Är den Azure-portalen hanterar samma som andra tjänster för CSP? 
Ja. Med StorSimple virtuell matris för CSP, ska det bästa sättet för partners åtkomst till Azure-portalen den [Partnercenter](http://partnercenter.microsoft.com/) där de kan hantera kunder och prenumerationer. Om du vill hantera StorSimple prenumerationer ska partner eller kund (beroende på behörighet) logga in på Azure-portalen. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Microsoft levererar en ny portal för StorSimple för CSP? 
Nej. Du kommer att kunna administrera virtuella StorSimple-matris på CSP via Azure-portalen.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Jag har etablerat en fristående StorSimple prenumerationen för min kunds. Måste jag använda Azure portal för att administrera min enhet? 
Ja. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Kompenseras Microsoft Account-team på grund av försäljningen av StorSimple via CSP?
Ja. Försäljningschef för partner och kunden kontoteamet för end-kund vill kompensera för CSP försäljning. Partners förväntas ta lead säljer sina differentierade värdet och erbjudanden.

## <a name="support-for-storsimple-and-csp"></a>Stöd för StorSimple- och CSP

### <a name="are-there-any-forums-to-get-additional-support-for-csp-partners-to-deploy-and-manage-storsimple"></a>Finns det några forum få ytterligare stöd för CSP-partner att distribuera och hantera StorSimple?
Du kan besöka StorSimple i CSP-forumet för att få svar på vanliga frågor. Du måste ansluta till den [Azure rådgivare Yammer grupp](https://www.yammer.com/azureadvisors) första. Nu söka efter och ansluta till gruppen - **StorSimple Partner rådgivare**.

### <a name="how-does-the-support-work-for-storsimple-for-csp"></a>Hur stödet fungerar för StorSimple för CSP? 
Support-modellen för StorSimple i CSP är samma som för andra Azure-tjänster i CSP. Mer information finns på [kundsupport för CSP](https://msdn.microsoft.com/partner-center/customer-support). 

Mer information om StorSimple för CSP finns:

- [Microsoft Cloud Solution Provider Program](https://partner.microsoft.com/Membership)
- [Partnercenter](http://partnercenter.microsoft.com/)
