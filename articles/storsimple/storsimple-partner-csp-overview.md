# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>Vad är StorSimple för molnet lösningar Providers Program?


## <a name="overview"></a>Översikt

Microsoft Azure StorSimple är en unik metod för företagslagring med true hybrid cloud lagringskapacitet. Det ger kunder att dra nytta av ekonomiska molnlagring för inaktiva data, samtidigt som deras verksamhetskritiska data lokalt för de högsta möjliga prestanda. 

StorSimple virtuell matris för molnet lösningar Provider (CSP) gör det möjligt för partner att utnyttja denna möjlighet. Partners äger livscykeln för slutpunkt till slutpunkt kunden med direkta etablering, fakturering och stöd för Microsofts molntjänster. Kort sagt, transact partners nu StorSimple tillsammans med sina tjänster till kunder.

Mer information om StorSimple för CSP finns i [översikt över Azure CSP](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview) sidan.

Mer information om fakturering prissättning, uppmuntra, och få support i CSP, gå till [StorSimple i CSP: vanliga frågor och svar](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Distribuera och hantera StorSimple för Kryptografiprovider

StorSimple för CSP är tillgänglig som en användningsbaserad tjänst för alla marknader där den virtuella StorSimple finns idag. StorSimple för CSP använder Azure-portalen och StorSimple enheten Manager-tjänsten. En CSP-partner kan skapa en StorSimple Device Manager för att hantera virtuella StorSimple-matriser, resurser, volymer och säkerhetskopieringar. Du kan administrera alla virtuella matriser som har registrerats med din StorSimple Device Manager-tjänsten via Azure portal.

Mer information finns på [distribuera och hantera din virtuella StorSimple-matrisen för CSP](storsimple-partner-csp-deploy.md).
