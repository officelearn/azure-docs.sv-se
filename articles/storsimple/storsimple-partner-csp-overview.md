# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>Vad är StorSimple för Cloud lösningar Providers programmet?


## <a name="overview"></a>Översikt

Microsoft Azure StorSimple är en unik metod för företagslagring till SANT lagring för hybridmolnsfunktioner. Det hjälper kunderna att dra nytta av ekonomisk molnlagring för inaktiva data samtidigt som deras verksamhetskritiska data lagras lokalt på högsta möjliga prestandanivå. 

StorSimple Virtual Array för molnet Molnlösningsleverantör (CSP) gör det möjligt för partner att utnyttja denna möjlighet. Partner som äger slutpunkt till slutpunkt-kundlivscykeln med direkt etablering, fakturering och support av Microsofts molntjänster. Kortfattat kan partner som nu transact StorSimple tillsammans med sina tjänster till kunder.

Mer information om StorSimple för CSP finns den [översikt över Azure CSP](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview) sidan.

Mer information om fakturering, prissättning, incitament, och få support i CSP, gå till [StorSimple i CSP: VANLIGA FRÅGOR OCH SVAR](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Distribuera och hantera StorSimple för CSP

StorSimple för CSP är tillgängligt som en användarbaserad tjänst på alla marknader där StorSimple är tillgängliga idag. StorSimple för CSP: N använder Azure-portalen och StorSimple Device Manager-tjänsten. CSP-partner kan skapa en StorSimple Device Manager för att hantera StorSimple Virtual Array, resurser, volymer och säkerhetskopior. Du kan administrera alla virtuella matriser som har registrerats för din StorSimple Device Manager-tjänsten via Azure portal.

Mer information går du till [distribuera och hantera din StorSimple Virtual Array för CSP](storsimple-partner-csp-deploy.md).
