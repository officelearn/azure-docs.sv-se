# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Använda tjänsten StorSimple Device Manager för att felsöka StorSimple Virtual Array
## <a name="overview"></a>Översikt

StorSimple-Enhetshanteraren ger en **diagnos och löser problem** i bladet för tjänst sammanfattning, som visar några av de vanligaste problemen som kan uppstå med den virtuella matrisen och hur du kan lösa dem. I den här kursen får du en introduktion till fel söknings funktionen i StorSimple Enhetshanteraren-tjänsten.

![Diagnostisera och lösa problem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnostisera och lösa problem

Du kan visa några av de vanliga problemen med den virtuella StorSimple-matrisen och granska lösningarna på problemen direkt från bladet StorSimple Enhetshanteraren service Summary.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Diagnostisera ett problem med din virtuella matris

1. Klicka på **diagnostisera och lös problem** inställning i den vänstra rutan i bladet StorSimple Enhetshanteraren tjänst sammanfattning om du vill visa en lista över vanliga problem.

2. **Expandera** på problemet som du påträffar för att granska **rekommenderade steg** som hjälper dig att felsöka problemet. Om du vill kan du också gå igenom de detaljerade dokumentations länkarna som finns för ytterligare referens.
   
    ![Felsöka ett problem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Om du inte kan hitta en referens till problemet eller lösa det, kan du kontakta Microsoft Support för ytterligare hjälp.
