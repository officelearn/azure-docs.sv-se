# <a name="use-the-storsimple-device-manager-service-to-log-a-support-request-for-the-storsimple-virtual-array"></a>Använda StorSimple Device Manager-tjänsten för att logga en supportförfrågan för StorSimple Virtual Array

## <a name="overview"></a>Översikt

StorSimple Device Manager ger möjlighet att **logga en ny supportbegäran** inom sammanfattningsbladet för tjänsten. Den här artikeln förklarar hur du kan logga en ny supportbegäran och hantera livscykeln från portalen.

## <a name="new-support-request"></a>Ny supportbegäran

Beroende på din [supportavtal](https://azure.microsoft.com/support/plans/), kan du skapa supportbegäranden för ett problem på din StorSimple Virtual array direkt från sammanfattningsbladet för StorSimple Device Manager-tjänsten.

#### <a name="to-log-a-new-request"></a>Logga en ny begäran

1. Gå till StorSimple Device Manager-tjänsten. I bladet sammanfattning tjänstinställningar går du till **stöd + felsökning** avsnittet och klicka sedan på **ny supportbegäran**.
   
    ![Ny supportbegäran](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket1.png)

2. I den **grunderna** bladet gör du följande:

    1. Från den **typ av problem** listrutan, väljer **teknisk**. 
    
    2. Aktuellt **prenumeration**, **Service** typ, och **Resource** (StorSimple Device Manager-tjänsten) väljs automatiskt. 

    3. Ange en eller flera enheter som registrerats till tjänsten som har problem.

    4. Välj ett lämpligt **supportavtal** om du har flera prenumerationer som är associerade med prenumerationen. Du behöver en plan för betald support för att teknisk Support.

3. I **steg 2**, Välj den **allvarlighetsgrad** och ange om problemet är relaterat till matrisen eller StorSimple Device Manager-tjänsten. Välj även en **kategori** för den här utfärda och ger mer **information** om problemet.
   
    ![Ny supportbegäran](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket2.png)

4. I **steg3**, ange din kontaktinformation. Microsoft Support använder den här informationen för att kontakta dig för ytterligare information, diagnos och lösning.
   
    ![Ny supportbegäran](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket3.png)

## <a name="manage-a-support-request"></a>Hantera supportärenden

När du har skapat ett supportärende kan du hantera ärendets livscykel på portalen.

#### <a name="to-manage-your-support-requests"></a>Hantera dina supportärenden

Gå till sidan Hjälp och support genom att gå till **Bläddra > hjälp + support**.

![Hantera supportförfrågningar](./media/storsimple-virtual-array-log-support-ticket/manage-support-tickets.png)
