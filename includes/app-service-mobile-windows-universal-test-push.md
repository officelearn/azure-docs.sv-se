
1. Högerklicka på Windows Store-projektet klickar du på **Set as StartUp Project**, tryck på F5 för att köra Windows Store-app.
   
    När appen startar är enheten registrerad för push-meddelanden.
2. Stoppa Windows Store-appen och upprepa det föregående steget för Windows Phone Store-app.
   
    Båda enheterna är nu registrerad för att ta emot push-meddelanden.
3. Kör Windows Store-app igen och ange text i **infoga en TodoItem**, och klicka sedan på **spara**.
   
       Note that after the insert completes, both the Windows Store and the Windows Phone apps receive a push notification from WNS. The notification is displayed on Windows Phone even when the app isn't running.
