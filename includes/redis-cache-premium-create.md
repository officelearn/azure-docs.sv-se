Om du vill skapa en premium-cache, logga in på den [Azure-portalen](https://portal.azure.com) och klicka på **skapa en resurs** > **databaser** > **Azure Cache för Redis**.

![Skapa ett cacheminne](media/redis-cache-premium-create/redis-cache-new-cache-menu.png)

> [!NOTE]
> Förutom att skapa cacheminnen i Azure Portal, kan du också skapa dem med hjälp av Resource Manager-mallar, PowerShell eller Azure CLI. Läs mer om hur du skapar en Azure Cache för Redis [skapa ett cacheminne](../articles/azure-cache-for-redis/cache-dotnet-how-to-use-azure-redis-cache.md#create-a-cache).
> 
> 

Om du vill konfigurera premiumfunktioner så måste du först välja en av premium-prisnivåerna på den **prisnivå** listrutan. För mer information om varje prisnivå, klickar du på **Visa fullständiga prisuppgifter** och välj en prisnivå från den **Välj din prisnivå** bladet.

![Välj din prisnivå](media/redis-cache-premium-create/redis-cache-premium-pricing-tier.png)
