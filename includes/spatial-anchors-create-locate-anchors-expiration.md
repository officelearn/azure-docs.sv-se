## <a name="set-expiration"></a>Ange förfallo datum

Det är också möjligt att konfigurera ankaret så att det upphör att gälla automatiskt vid ett visst datum i framtiden. När ett ankare upphör att gälla kommer den inte längre att finnas eller uppdateras. Förfallo datum kan bara anges när ankaret skapas. Det går inte att uppdatera förfallo datum efteråt. Så du kan ange dess förfallo datum innan du sparar den i molnet.
