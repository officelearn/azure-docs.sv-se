När du arbetar med den klassiska distributions modellen kan du inte använda Azure Cloud Shell. I stället måste du installera den senaste versionen av Azure Service Management (SM) PowerShell-cmdlets lokalt på datorn. Dessa cmdletar skiljer sig från AzureRM-eller AZ-cmdletarna. Information om hur du installerar SM-cmdlets finns i [Installera Service Management-cmdletar](/powershell/azure/servicemanagement/install-azure-ps). Mer information om Azure PowerShell i allmänhet finns i Azure PowerShell- [dokumentationen](/powershell/azure/).
